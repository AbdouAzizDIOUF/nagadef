$strictFederatedSTS

AbstractSettings.php : 157

Whether to use the literal value of STSAuthURL in GetUserRealm response.

public bool $strictFederatedSTS = false

The toolkit only supports one ADFS endpoint, /adfs/services/trust/13/usernamemixed. In non-standard environments, STSAuthURL may point to a different endpoint. The default behavior is to always point to the UsernameMixed endpoint. This setting allows to override such behavior and instead use the exact presented value.
